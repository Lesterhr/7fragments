# Die 8 Fragmente

Abschluss des AI Trainings (Grundlagen, Sessions 1 bis 5) als Browser-Minigame.

## Spiel herunterladen und starten

**[Download: Die_7_Fragmente.html](https://github.com/Lesterhr/7fragments/releases/latest/download/Die_7_Fragmente.html)**

1. Datei über den Link oben herunterladen
2. Die heruntergeladene Datei per Doppelklick öffnen (Chrome oder Edge empfohlen)
3. Namen eintragen und losspielen

Keine Installation, kein Server: Das Spiel ist eine einzige HTML-Datei und läuft komplett lokal im Browser. Die benötigten Anhänge (Besprechungsprotokoll, Messdaten) sind im Spiel eingebettet und lassen sich an den jeweiligen Stationen per Button herunterladen.

## So funktioniert es

Auf der Karte warten 8 Stationen mit Rätseln aus dem Training. Für jedes gelöste Rätsel gibt es ein Fragment, das zum Sockel im Tempel der Fragmente getragen wird. Sind alle 8 Fragmente eingesetzt, wird das Zertifikat freigeschaltet. Über "PDF erstellen" entsteht die Abgabedatei: Zertifikat plus detaillierter Antwortbericht (im Druckdialog "Als PDF speichern" wählen).

Steuerung: Pfeiltasten oder WASD, alternativ Klick auf die Karte. Taste E öffnet ein Rätsel in der Nähe.

Hinweis: Für die Original-Schriftarten wird eine Internetverbindung benötigt; ohne Verbindung läuft das Spiel mit Ersatzschriften trotzdem vollständig.
